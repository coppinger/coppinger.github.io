The Difference Between Inline and Inline-Block

Explaining the difference between inline and inline-block in CSS

When it comes to the foundational building blocks of web design, HTML and CSS, once you've discovered how to add basic elements such as div's and text, the next step is to figure out how to display them.

The core of layout in CSS comes down to a handful of properties:


  • display: inline;
  • display: block;
  • display: inline-block;
  • display: none;

display: inline;


The display: inline; property tells an element that should sit on the same line as other elements. This also means that inline elements cannnot have a width and height set, and do not respect top and bottom margin or padding.


display: block;


The display: block; property tells an element that it's a block, and as such, it should force a line break on the line that it is on, so that other elements won't sit next to it, but rather fall to the line below. This can be observed in the diagram to the right.


display: inline-block;


The display: inline-block; property tells an element to behave like it has display: inline; property, but that it should also allow it to have a specific width and height, like a display: block; element.
